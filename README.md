### About

I'm **Deamble Lopes**, based in **Maringá, PR, Brazil**. Open to new opportunities.

**Specialties**

- **Backend & APIs** — Laravel / PHP, REST-style endpoints, validation, auth patterns (e.g. Sanctum), queues and scheduled work when the domain needs it  
- **Data** — MySQL modeling, migrations, relationships, and queries tuned for real business rules  
- **Web UI** — Blade, Vite, Tailwind CSS, JavaScript / TypeScript; pragmatic UX for internal and customer-facing apps  
- **Platform** — Docker, Linux, Git; repeatable environments and shell automation (Bash / PowerShell)  
- **Polyglot when it fits** — Node.js and Python for tooling, scripts, or services alongside the main stack  

**Private work (examples)**

Delivery I’ve owned in **private and client repositories** (code stays off the public profile).

- Laravel applications: authenticated modules, domain rules, exports, and integrations with internal systems or external APIs  
- MySQL-heavy work: transactional flows, reporting-oriented queries, and careful schema evolution across releases  
- Docker/Linux runtimes, Git-based collaboration, and release-friendly workflows with the team
